## Getting Started

Clone this project.
```
$ git clone https://github.com/hfvaldesg/kurve.git
```

Install all the dependencies:
* GLFW
* FreeType2

Build the project
```
$ cd kurve
$ mkdir build
$ cd build
$ cmake ..
$ make
```

Enjoy
```
./kurve
```
